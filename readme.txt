A beautiful, interactive mood slider for your WordPress site.

== Description ==

Mood Slider is a premium interactive component that allows users to express how they feel by sliding a thumb. The component updates text, descriptions, and call-to-action buttons dynamically.

Key Features:
* Interactive Mood Slider with smooth transitions.
* Elementor Widget with full styling controls.
* Shortcode support: `[mood_slider]`.
* Global settings for colors and content.
* Premium design aesthetics.

== Installation ==

1. Upload the `mood-slider-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Mood Slider to configure global colors.
4. Use the `[mood_slider]` shortcode or the Elementor widget to display the slider.

== Frequently Asked Questions ==

= How do I use the shortcode? =
Simply add `[mood_slider]` to any page or post. You can also use `title` and `subtitle` attributes.

= Does it support Elementor? =
Yes, look for the "Mood Slider" widget in the Elementor editor.
